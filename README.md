# Magarete Njira | Portfolio

This is a [Next.js](https://nextjs.org) portfolio website for Magarete Njira, a Fullstack Developer.

## Features

- **Next.js 15+** with App Router
- **TypeScript** for type safety
- **Tailwind CSS** for styling (with legacy styles integrated)
- **Responsive Design** for all devices
- **Project Filtering** functionality
- **Contact Form** integration via FormSubmit

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app/`: Next.js App Router pages and global styles.
- `src/components/`: Reusable React components for each section of the website.
- `public/`: Static assets including images and PDF files.

## Build and Deployment

To create an optimized production build:

```bash
npm run build
```

The build output will be in the `.next` folder.
